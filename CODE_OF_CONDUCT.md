# 行为准则 / Code of Conduct

Spec Bifrost 社区希望保持专业、友善和以问题解决为中心。

The Spec Bifrost community aims to stay professional, welcoming, and focused on solving problems.

## 我们鼓励 / We Encourage

- 尊重不同经验背景和表达方式。
- 用事实、复现和权衡讨论技术问题。
- 给出具体、可执行的反馈。
- 保护用户、客户和公司的敏感信息。

- Respect different backgrounds and communication styles.
- Discuss technical topics with facts, reproduction steps, and tradeoffs.
- Provide specific, actionable feedback.
- Protect sensitive user, customer, and company information.

## 不接受 / Not Accepted

- 人身攻击、骚扰、歧视或威胁。
- 发布 secrets、私有配置或真实客户数据。
- 反复偏离项目边界的争论。
- 垃圾信息或恶意破坏。

- Personal attacks, harassment, discrimination, or threats.
- Publishing secrets, private configuration, or real customer data.
- Repeated arguments outside the project scope.
- Spam or malicious disruption.

## 处理方式 / Enforcement

维护者可以编辑、隐藏或删除违反本准则的内容，也可以限制相关账号参与项目。

Maintainers may edit, hide, or remove content that violates this code, and may restrict the related accounts from participating in the project.

如需报告问题，请通过 GitHub issue 或仓库 owner 的公开联系方式联系维护者。安全漏洞请遵循 `SECURITY.md`。

To report a concern, contact the maintainers through GitHub issues or the repository owner's public contact channels. For security vulnerabilities, follow `SECURITY.md`.
